# Simple Platformer Game

A simple HTML5 Canvas platformer game with one level.

## How to Play

1. Open `index.html` in your web browser
2. Use **Arrow Keys** or **WASD** to move:
   - Left/Right: Move horizontally
   - Up/W: Jump
3. Navigate through the platforms
4. Reach the green flag at the top right to win!

## Features

- Player character with physics-based movement
- Multiple platforms to jump on
- Gravity and collision detection
- Goal flag to reach
- Win condition with restart option

## Controls

- **Arrow Keys** or **WASD**: Move and jump
- **Spacebar**: Jump (alternative)

## Technical Details

- Built with HTML5 Canvas and vanilla JavaScript
- No external dependencies required
- Responsive physics system with gravity and friction
- Collision detection for platforms and boundaries

Enjoy the game!
